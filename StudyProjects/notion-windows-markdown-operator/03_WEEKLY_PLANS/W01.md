## Active Recall Questions

Start every day with 5 questions:

1. What did I learn yesterday that changes how I operate today?
2. Which shortcut or workflow saved time yesterday?
3. Which step created friction?
4. What did I overbuild or underbuild?
5. What is the smallest useful output today?

## Calibration Rules

| End-of-day finding | Next-day change |
|---|---|
| Too much theory | Replace 50% of lesson with timed drills |
| Too many Notion objects | Consolidate before adding anything |
| Low recall | Review before new content |
| High friction in Windows | Repeat D1 drills for 20 minutes |
| Strong performance | Add one real work/learning scenario |
