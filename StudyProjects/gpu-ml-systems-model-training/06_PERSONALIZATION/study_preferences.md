# Study Preferences - GPU ML Systems & Model Training

Last updated: 2026-06-14

## Known User Requirements

- 中文讲解为主，保留必要英文专业术语。
- 必须先讲懂概念，再做实操；不能把搭建工具当学习本身。
- 每个概念都要回答：是什么、机制是什么、怎么用、如何提升能力/生产力/判断力、ROI 最大化行动是什么。
- 用户对模型置信度和 agent 自主执行不信任，要求外部硬约束、预算边界、状态机、日志和可验证规则。
- 用户对“Top1%”要求高，低密度笔记会被视为无效。

## Teaching Adjustments

- 每天先用主动回忆检查上一次核心概念。
- 讲解必须包含失败模式和反例。
- 实验必须先写 run contract：目标、预算、输入、命令、验证、停止条件。
- 遇到不确定硬件/版本，不猜，先 profile 或查官方文档。
