# Study Preferences

## Current Requirements

| Preference | Rule |
|---|---|
| Language | 中文为主，专业术语保留英文。 |
| Style | 深讲概念，不只给问题。 |
| Goal | 每个概念都要解释如何使用，如何提高能力、生产力、思维方式、生产效率。 |
| ROI | 所有学习都要转成现实行动、Codex/Agent 工作流或成本治理规则。 |
| Format | 表格优先，结论清晰，避免泛泛而谈。 |

## Teaching Rule

每天必须回答：

1. 这个概念是什么。
2. 机制是什么。
3. 我应该怎么用。
4. 它如何影响 AI agent、Codex、本地模型或成本治理。
5. 最小高 ROI 行动是什么。
