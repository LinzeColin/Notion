# Study Preferences

## Stable Preferences

| Preference | Project application |
|---|---|
| 中文为主 | 学习笔记、Notion 内容、daily logs 默认中文。 |
| ROI 最大化 | 每天必须留下一个可复用研究资产。 |
| 概念先行 | 不先做股票筛选或工具搭建；先讲透概念、机制、边界。 |
| 低伪进度 | 不用大量文件数量伪装进度；用可验证输出判断进展。 |
| Notion 清洁 | Notion 是轻量笔记和可视化，不污染原始 workspace。 |
| GitHub 长期记忆 | GitHub 保存 state、logs、计划、同步记录和 handoff。 |

## Teaching Fit

讲解每个核心概念时必须回答：

1. 这个概念是什么。
2. 这个概念的核心机制是什么。
3. 这个概念如何帮助我理解生活、决策、学习和工作。
4. 如何把这个概念转成能力、生产力、思维方式或生产效率。
5. 从 ROI 最大化角度，今天最小行动是什么。
6. 如何验证我真的理解，而不是只记住词。
