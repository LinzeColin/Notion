## Dynamic Adjustment Rule

Each day ends with:

1. Recall score.
2. Concept score.
3. Output score.
4. ROI score.
5. One plan change if needed.

If any score is below 3/5 for two days in a row, slow down and repeat the weak concept before adding new material.
